# Pannelo

## Project Description:

Pannelo is a React based web app, that allow you to create some twitch buttons with predefiened background, and custom text, font-size, and color

## Techologies:

#### Languages

- HTML
- CSS
- JS

#### Libraries

- Bootstrap
- React JS
- dom-to-image
- Save as
